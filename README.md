# Morse_Code_Generator
The Morse Code Generator is a Java program that converts text into Morse code and vice versa. It utilizes a HashMap to store Morse code equivalents of characters, providing a simple and efficient conversion process. The program offers a menu-driven interface, allowing users to easily choose between converting text to Morse code or Morse code to text.
